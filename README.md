Customer Shopping Behaviour Analysis
📌 Project Overview

This project analyzes customer shopping behavior to uncover insights into purchasing patterns, customer segmentation, product performance, and revenue trends. It combines SQL, Python (Jupyter Notebook), and Power BI to perform end-to-end data analysis and visualization.

The goal is to help businesses make data-driven decisions by understanding how customers interact with products, discounts, subscriptions, and shipping options.

📂 Project Files
customer_shopping_behavior.csv → Raw dataset
customer_behaviour_analysis_sql.sql → SQL queries for analysis
customer_behaviour.ipynb → Data analysis using Python
Customer Behaviour Dashboard.pbix → Interactive Power BI dashboard

🧰 Tools & Technologies Used
Python (Pandas, NumPy, Matplotlib/Seaborn)
SQL (MySQL) for querying and insights
Power BI for dashboard visualization
Jupyter Notebook for exploratory data analysis
📊 Key Analysis Performed

Compared total revenue generated by male vs female customers
Analyzed revenue contribution by age group

Identified customers who used discounts but still spent above average
Segmented customers into:
New
Returning
Loyal customers
🔹 Product Insights
Top 5 products with highest average ratings
Top 3 most purchased products per category
Products with highest discount usage rate
🔹 Subscription Insights
Compared average spending and revenue between:
Subscribed customers
Non-subscribed customers
Checked if repeat buyers are more likely to subscribe
🔹 Shipping Analysis
Compared average purchase amount between:
Standard Shipping
